Diary

Your private, encrypted journal — fully offline.

What it does

  • Daily entries — rich text, mood (1–5), tags, date.
  • Photos & videos — attach to any entry. Encrypted on device.
  • Voice notes — tap 🎤 in the editor to record (max 5 min).
  • Calendar view — month grid; mood-tinted day cells.
  • On this day — entries from the same date in past years surface on the timeline.
  • Search — instant prefix search across titles and bodies. Persistent index.
  • Tags — tap a tag chip to filter the timeline.
  • Trash — deleted entries are soft-deleted; auto-purged after 30 days. Restore anytime.
  • Backups — encrypted .json file export + 3 auto-rolling snapshots stored locally.
  • Dark / light mode — toggle ☀/🌙 in any header.
  • PWA — install to home screen, runs offline.
  • Master code — your signature "1810" works as a backup unlock alongside your passcode.

How to use

1.
Create an entry

Tap + in the bottom nav. Add title, pick a mood, write. Tap Save.

2.
Attach media

+ Photo / Video picks from library (also accepts audio files). 📷 Camera opens the camera. 🎤 Record starts a voice note — tap again to stop.

3.
Browse

Use List for chronological view, Calendar for month overview. Tap a calendar day to open that entry, or to start a new one pre-filled with that date. Tap the month title to jump back to today.

4.
Search & filter

Type in the search bar — results update as you type. Tap any tag chip on a row to filter the timeline by that tag; tap the pink filter pill to clear.

5.
Trash & restore

Deleting an entry moves it to Settings → Trash. You can restore it anytime within 30 days. After 30 days it's permanently purged.

6.
Back up

Settings → Encrypted Backup downloads a .json file. Save it somewhere safe (cloud drive, AirDrop). The app also keeps the latest 3 snapshots locally — see Settings → Local Backups to restore from them or take one on demand.

7.
If your device dies

Open Diary on a new device, set up with your passcode, then go to Settings → Restore Backup and pick the file. All entries, photos, videos, and voice notes return.

8.
Install on iPhone

Open in Safari → Share → Add to Home Screen. Or tap Install on the lock-screen banner if it appears. Runs full-screen, fully offline.

Privacy & security

  • Local-only. No server, no cloud, no analytics. Everything stays on this device.
  • Encrypted. Entries, photos, videos, voice notes, thumbnails — all AES-GCM 256-bit.
  • Two-key unlock. Your passcode unlocks the diary. The master code 1810 also works as a personal backup unlock.
  • No recovery. Forget both codes and lose your backup file? Data is gone forever. Keep at least one backup off-device.
  • Auto-lock. App locks after 2 minutes in background.
  • Backup safety. Restoring a backup validates the file (and verifies its sha256 checksum) BEFORE replacing your current data — wrong file won't wipe your diary.

Tips

  • Drafts auto-save every 3 seconds while editing — close the tab safely.
  • Word count and reading time appear under each entry on the detail page.
  • Header tools (☀/🌙 + ⓘ) are on every screen — theme toggle and this Help screen are always one tap away.
  • Hard refresh: tap ↻ next to the version on the lock screen to fetch the latest update.
  • If a new version is available while you're using the app, a pink banner appears at the top — tap Reload.
